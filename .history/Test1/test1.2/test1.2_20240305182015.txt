Interface เป็นเหมือนกล่องเครื่องมือที่บอกว่าสิ่งนี้สามารถทำอะไรบ้าง โดยไม่ได้ให้คำสั่งเฉพาะแต่อย่างใด ๆ ซึ่งให้โค้ดอื่น ๆ มาสร้างฟังก์ชันหรือคุณสมบัติให้เป็นไปตามที่กำหนดไว้ใน Interface นั้น ๆ อย่างเช่น Interface ที่เป็นกล่องเครื่องมือสำหรับรถยนต์อาจมีเมทอดคือ "เร่ง", "เบรค", หรือ "เปิดไฟสว่าง" แต่ไม่ได้ให้คำสั่งเฉพาะไว้ใน Interface นั้น เมื่อคลาสใดก็ตามที่ต้องการใช้ Interface นี้ จะต้องมาสร้างโค้ดของฟังก์ชันเหล่านี้ให้ถูกต้องตามที่ Interface กำหนดไว้

Abstract class เป็นเหมือนแม่แบบหรือโครงสร้างพื้นฐานที่ถูกกำหนดไว้ล่วงหน้า โดยส่วนที่ยังไม่ชัดเจนหรือไม่สมบูรณ์ก็จะถูกประกาศใน Abstract class นั้น และมันสามารถมีเมทอดที่เป็น abstract ได้ ซึ่งเมทอดเหล่านั้นจะต้องถูกการทำงานเป็นรายการโค้ดในคลาสที่สืบทอด (inherit) มาจาก Abstract class นั้น ๆ ไม่ว่าจะเป็นการสร้างฟังก์ชันหรือการกำหนดคุณสมบัติที่ไม่สมบูรณ์ให้กับโค้ดของคลาสที่สืบทอดนั้น ๆ
