5.1 Responsive Web Design:
เว็บไซต์ปรับขนาดให้เหมาะกับทุกอุปกรณ์
สะดวกต่อผู้ใช้ ประหยัดเวลาและค่าใช้จ่าย

5.2 RESTful API:
สื่อสารกับข้อมูลโดยใช้ JSON
มีน้ำหนักเบา อ่านง่าย รองรับภาษาโปรแกรมหลากหลาย

5.3 HTTP Method:
GET: ดึงข้อมูล
POST: สร้างข้อมูล
PUT: แก้ไขข้อมูล
DELETE: ลบข้อมูล

5.4 JavaScript vs TypeScript:
JavaScript: ไดนามิก ไม่ต้องประกาศตัวแปร เหมาะกับงานง่าย
TypeScript: superset ของ JavaScript รองรับการประกาศตัวแปร เหมาะกับงานใหญ่

5.5 Server-side vs Client-side:
Server-side: ประมวลผลบนเซิร์ฟเวอร์ ปลอดภัย เหมาะกับงานหนัก
Client-side: ประมวลผลบนเว็บเบราว์เซอร์ ตอบสนองเร็ว เหมาะกับงานเบา
